A project developed based on the [curso.dev](https://curso.dev) by Filipe Deschamps.

## Technologies Used

## Installation

1. Clone the repository:
   ```bash
   git clone [repository-url]
2. Navigate to the project directory:
   ```bash
   cd clone-tabnews
3. Install the dependencies:
   ```bash
   npm install
3. Run application:
   ```bash
   npm run dev
